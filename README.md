# DataLens - Albergue Central Bogotá

Sistema de análisis inteligente para la gestión de habitantes y recursos financieros utilizando Gemini IA.

## Estructura del Proyecto
- **/frontend**: Interfaz de usuario (HTML5, CSS3, JS).
- **/backend**: Lógica de procesamiento en Python y conexión con IA.

## Instalación
1. Clonar el repositorio.
2. Instalar dependencias: `pip install -r requirements.txt`
3. Crear un archivo `.env` en la raíz con tu clave: `GEMINI_API_KEY=tu_llave_aqui`

## Uso
1. Ejecutar el backend: `python backend/main.py`
2. Abrir los archivos HTML desde la carpeta `/frontend`.
